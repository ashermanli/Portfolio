Phonebook App

Following along with the full stack course provided by fullstackopen.com, I have been learning to create single page web applications using the MERN stack. I have created the front-end using Reactjs to display an interface where existing phonebook entries are shown as well as a form that lets you input new entries. The back-end uses an Express server running on the NodeJS runtime environment, and it handles the connection to a MongoDB database and the operations it performs such as the creation or update of an entry, or the deletion of an entry.
